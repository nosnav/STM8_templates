configure GPIO pin

Parameters
[in]addrPortaddress of port to configure, e.g. PORT_A (see stm8as.h)
[in]pinsbitmask of one or more port pins to configure, e.g. PIN_1 (see gpio.h)
[in]configselect pin configuration, e.g. INPUT_PULLUP_NOEXINT (see gpio.h)

configure GPIO port pins as input or output with different features activated. For supported configurations see gpio.h. Multiple port pins can be configured with same configuration
