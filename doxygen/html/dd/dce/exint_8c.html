void exint_init ( GPIO_TypeDef *  addrPort,
uint8_t  edge 
)

edge control for EXINTs

Parameters
[in]addrPortaddress of port to configure, e.g. PORT_A (see gpio.h)
[in]edgeselect edge for exint, e.g. EXINT_RISE_EDGE (see gpio.h)

configure edge control for GPIO external interrupts. Can only be done for complete port. For pins with EXINT functionality see exint.h.

Note that all interrupts have to be disabled when calling this routine!
